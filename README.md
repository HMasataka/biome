# Biome

## Getting Started

```bash
npm install --save-dev @biomejs/biome
```

```bash
npx biome init
```

```bash
CocInstall coc-biome
```

## Run

```bash
npx biome format ./src
```

```bash
npx biome format --write ./src
```

```bash
npx biome lint ./src
```

```bash
npx biome lint --apply ./src
```
